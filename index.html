📝 Syntax Guide

Instructions (col J)
One per line. ** prefix = bold. [TABLE] = inline data table.
Verify area is clear of obstructions. **Confirm area is safe for drum handling. Check reactor manway is open. [TABLE] Material Classification Material | Type | Status DMI-001 | API | {restricted}Restricted SOL-002 | Solvent | {unrestricted}Unrestricted EXC-003 | Excipient | {unrestricted}Unrestricted Instructions continue after the table.
[TABLE] title on first line. Next pipe-delimited line = headers. Subsequent pipe lines = rows. {restricted} or {unrestricted} prefix = red/green cell styling. A non-pipe line ends the table.
Parameters (col K)
Input: just the label. Select: label|options. * = required.
*Scan Material Barcode ← required text input *Gross Weight (kg) ← required numeric input Comments ← optional text input *Select Action|Re-scan,Abort ← required dropdown *Location|Disp A,Disp B,Cold Store ← required dropdown
Signatures (col L)
Bracket suffix = popup role.
Operator [Op] ← single sign, operator popup Op + Verifier [Ver] ← dual sign, verifier popup Op + Supervisor [Sup] ← dual sign, supervisor popup
Actions (col M)
Nav links, section headers, error loop-backs.
Submit Scan → MC_NEXT ← navigation link [S] Supervisor Decision ← bold section header Continue → MC_PROCEED ← navigation link Abort → MC_END_ERR ← navigation link [RETURN] MC_SCAN ← error loop-back (amber)
Notes (col N)
Free text. Error: prefix = popup message.
Req: GCSP-40 CO: OP_EQUIP_SCAN_CHECK_1 Alias: MCF_DS_SKIDPREP_START QR: 2 Executed By: Op Error: Material does not match BOM. ← error popup (Evidence Required)
BoS / Hazard / SOP (cols O–Q)
Indicator badges displayed as links in top-right corner of wireframe.
BoS (col O): Y or N ← red "Basis of Safety" link Hazard (col P): Y or N ← amber "Hazard Statement" link SOP (col Q): SOP-12345 ← blue "SOP: SOP-12345" link (blank) ← hidden
Next (col D) — Routing
MC_SCAN_DRUM ← sequential Y:MC_WEIGHT | N:MC_ERR ← decision || MC_DCS, MC_MES, MC_LOG ← parallel split >> MC_SCAN_DRUM ← loop back Y:MC_NEXT | N:>> MC_PREV ← decision + loop Re-write:>> MC_WRITE | Manual:MC_MANUAL ← SELECT branches (Step_Type=SELECT) ← blank = END
ISA-88 Instances (cols F–G)
UP_EQUIP_CHECK:1 ← 1st UP instance UP_EQUIP_CHECK:2 ← 2nd UP instance OP_SCAN_EQUIP:1 ← 1st scan operation OP_GET_CLASS:3 ← 3rd classification
